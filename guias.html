Tank Headlights / Game Freezing Issue

Some tanks include custom headlights. Because the game is not designed to remove 2D styles, decals, or paint jobs from models that include custom lights, conflicts may occur.

To avoid this issue, it is recommended to remove any 2D styles or visual customizations before installing the skin.

It's not mandatory; do it only if you want to remove a 2D style from the tank.

Can't See Custom Tracks or Special Effects?

  1. Close the game completely.
  2. Download and install the required files here .
  3. Install [atacms]disable_persistent_data_cache.wotmod in the mods folder.
  4. Make sure the track physics are set to maximum in: Settings → Graphics → Details → Track Physics.
  5. if you used skins with special effects, install Special Effects Add-on.wotmod. otherwise it is not necessary

Tank Barrel Deformation Fix

If the barrel appears distorted, set Antialiasing to Maximum.

Settings → Graphics → Details → Antialiasing
